# 🧺 SpinCycle Laundry Management System

Aplikasi berbasis web untuk mengelola data laundry, mulai dari input pesanan, update status, hingga laporan.

## 🚀 Fitur Utama
- Login Admin
- Input Data Laundry
- Lihat Data Laundry
- Update Status Laundry
- Hapus Data
- Cek Status oleh Pelanggan
- Generate Laporan

## 🛠️ Teknologi
- PHP (CodeIgniter 4)
- MySQL
- Bootstrap

## 📌 User Story
Contoh:
- Sebagai admin, saya membutuhkan fitur input data laundry agar dapat mencatat pesanan pelanggan.
- Sebagai pelanggan, saya membutuhkan fitur cek status agar mengetahui progres cucian.

## ✅ Acceptance Criteria (Gherkin)
- Given admin berada di halaman login  
  When memasukkan data valid  
  Then masuk ke dashboard
